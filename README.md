## ☄️ Usage

Compiles and hot-reloads for development
```bash
npm run dev
```

Compiles and minifies for production
```bash
npm run build
```

### 🛠️ Technologies

The following tools were used in the construction of the project:

- [Vue.js](https://vuejs.org/)
- [Vuetify.js](https://vuetifyjs.com/)
